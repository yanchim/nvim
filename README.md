# Neovim configuration

Require latest stable neovim release, based on lua.

## INSTALL

For unix

    git clone https://github.com/yanchim/nvim.git $HOME/.config/nvim

For windows

    git clone https://github.com/yanchim/nvim.git $HOME/AppData/Local/nvim
